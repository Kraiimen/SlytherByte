## Authentication
<p>
  The authentication page includes both the login and registration forms. <br/>
  The user can switch between them using the “<b>Create new account</b>” or “<b>Already have an account?</b>” links. <br/>
  When the user successfully registers, the login form is shown automatically.
</p>

<p>
  When the user enters their email, a username is automatically generated. <br/> 
  The username is created by taking characters from the email address up to either the @ symbol or a maximum of 16 characters. <br/>
  The user can still edit the username manually if they wish.
</p>
